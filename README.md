# Palindrome Checker

App para verificar si una palabra o frase es palindroma en este proyecto podras probrar la funcionalidad como revisar el codigo

## Instalacion

Instala las dependencias del proyecto

```bash
npm install
# o si usas yarn
yarn install
```

## Ejecucion del proyecto

Modo de desarrollo

```bash
npm run dev
# o si usas yarn
yarn dev
```

Modo de Produccion

```bash
npm run build
npm start
# o si usas yarn
yarn build
yarn start
```
